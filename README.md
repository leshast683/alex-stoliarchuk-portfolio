# Alex Builds Web

🌐 [alexbuildsweb.com](https://alexbuildsweb.com)

A full-stack portfolio website showcasing my work in UX/UI design and web development.

## Projects

### SWC 2026 Summer Program
Designed and launched the official landing page for SWC's 2026 Summer Program — the primary hub for registration and program information.
**Tech:** Graphy

### Radio App Redesign
A full UX/UI redesign of a radio app, focused on cleaner navigation and a better mobile experience. Includes user research, wireframes, and high-fidelity mockups.
**Tech:** Figma, User Research, Prototyping

### Priority Manager App
A task management app design with scheduling tools, reminders, and progress tracking — built to simplify daily organization.
**Tech:** Figma, User Research, Prototyping

## Tech Stack

- React + Vite
- Framer Motion
- Express (contact form API)
- Resend (email delivery)
- Vercel (deployment)

## Run Locally

```bash
npm install
npm run dev
```
